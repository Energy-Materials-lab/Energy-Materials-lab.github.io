# Homepage of the Energy Materials Laboratory (EML)

This repository contains the source code for the official website of the **Energy Materials Laboratory (EML)** at the American University in Cairo. The site presents our research themes, publications, people, and news.

## Features

This website is designed for a research group homepage. Key features include:

* Clean, responsive design optimized for clarity and accessibility.
* Support for inline and display math equations via [MathJax](https://www.mathjax.org/).
* Syntax-highlighted code blocks for computational modeling and teaching materials.
* Structured pages for **Research**, **People**, **Publications**, and **News**.
* Atom/RSS feeds for updates and publications.
* SEO optimization via [jekyll-seo-tag](https://github.com/jekyll/jekyll-seo-tag).
* Automatic sitemap generation via [jekyll-sitemap](https://github.com/jekyll/jekyll-sitemap).

The site is deployed to GitHub Pages at:  
[https://eml-auc.github.io](https://eml-auc.github.io)  
(or via a custom domain if configured).

## Usage

If you wish to reuse this Jekyll theme for your own group:

1. Please give credit to the original theme authors (HPSTR, Clean Blog, TWiStErRob Blog).
2. Remove all personal materials and information related to EML.
3. Clear contents in the following directories: `_posts`, `_drafts`, `_data`, `_site`, `downloads`, `images`, and `pages`.
4. Modify `_config.yml` to add your own group information.
5. Add your content (research, people, publications, etc.).
6. Optionally, create per-category pages under `blog/category`.
7. Publish your site via GitHub Pages.

A sample blog post template can be found at `_posts/2020-04-13-post-template.md`.

## Important Notice

**Please remove all EML-specific content before publishing your own website using this theme.**
